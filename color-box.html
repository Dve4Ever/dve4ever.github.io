HTML5 Transparent Color-Box Custom Tag.

Made by MrDevoooo

How to Install

To install the color-box tag, you must paste in the following code in head section of your page:

<link rel="stylesheet" href="https://dve4ever.github.io/css/color-box-latest.css"> tag
